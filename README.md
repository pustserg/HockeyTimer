# HockeyTimer

A hockey shift timer app for iOS. Tracks player shifts with configurable intervals, audio/haptic alerts, and Live Activities support.

## Features

- Configurable shift intervals and game duration
- Shift tracking with visual countdown
- Sound and haptic alerts
- Live Activities & Dynamic Island integration
- Background timer support
- Customizable appearance (colors, backgrounds)

## Technologies

- SwiftUI
- ActivityKit (Live Activities & Dynamic Island)
- WidgetKit
- AVFoundation

## Requirements

iOS 16.1+
